mployee Burnout Intelligence: Organizational Health Analytics

Project ini menganalisis faktor-faktor "silent killer" penyebab kelelahan karyawan (burnout) yang berujung pada tingginya angka turnover. Menggunakan teknik Exploratory Data Analysis (EDA) dan Uji Statistik, analisis ini bertujuan memberikan insight berbasis data kepada HR dan Manajemen untuk mencegah talent drain dan meningkatkan produktivitas tim.

Key Focus:

Identifying correlation between work-setup (WFH/WFO) and mental fatigue.

Analyzing the impact of resource allocation on employee performance.

Providing actionable recommendations for retention policies.
